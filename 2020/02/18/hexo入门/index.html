hexo入门

环境准备

  1. Node.js. 选择LTS，长期支持版
  2. git

！配置git，不然后面步骤容易出错。

使用

初始化博客

hexo init 文件夹名称
新建文章
hexo new 文章名称
(或者)
hexo n 文章名称
在线预览
hexo server
(或者)
hexo s
一键部署
hexo deploy
(或者)
hexo d

hexo clean 清理
hexo g 生成
